2018-02-12
  • SVG 쓰면 깔끔하다 ??
  • 트리 구조를 이해하는게 파싱이고 구현하는게 렌더링이다.
  • 구글 크롬에 내장된 v8엔진은 램을 많이 사용한다.
  • 자바스크립트는 v8 전후로 나뉜다.
  • Edge 엔진도 괜찮다. 웹 표준도 잘 지킨다.
  • 인스턴트앱 : 앱을 깔지 않아도 앱처럼 동작한다.
  • 프로그레시브 웹앱(PWA) : 웹페이지를 앱처럼 보이게 한다.
  • 장고의 장점은 대시보드이며 노드는 다 구축해야 한다.
  • Travis CI, Jenkins : 배포 자동화, 테스트부터 다 해준다. ??
  • Celery는 장고에서 많이 쓴다.
  • Uri : 프로토콜, 도메인, 패스, 파일의 모든 정보를 알려준다.
  • URL : 파일 이름이 없다.
  • URN : 통신방법(프로토콜)이 상관 없다.
  • API(Application Programming Interface) : 응용 프로그램에서 사용할 수 있도록 운영체제나 프로그래밍 언어가 제공하는 기능을 제어할 수 있게 만든 인터페이스
  • Web API : 웹서버 혹은 웹 브라우저를 위한 API
  • REST API : REpresentational State Transfer API
    Resource - URI / Verb - HTTP method / Representations - 표현
    특징 : 범용성 / 리소스 중심 API 명세(URI를 읽는 것으로 이해 가능) / Stateless(클라이언트의 상태를 신경 쓰지 않음)
    Pros : 스펙없이 기존의 HTTP를 이용해 요청을 처리할 수 있다. ??
    Cons : 사용할 수 있는 메소드가 4개다. 표준이 없다.
  • HTTP Response code : 200, 201, 400, 404, 403, 500
