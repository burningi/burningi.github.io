1-19노트

2018-02-15
  • background-position : 50% 50%의 의미는 상자의 50/50과 그림의 50/50을 맞춘다는 의미다.
  • background-attachment : fixed페이지 스크롤을 내려도 배경을 고정시킨다.
  • 먼저 선언한 bg가 위로 깔린다.
  • 여백을 추가하는 방식은 성능을 낮춘다.
  • reflow repaint
  • tramsform 속성에 할당할 수 있는 값으로 trnaslate(), scale(), skew(), rotate() 함수가 있다.
  • css gpu animation은 성능이 더 좋다.
  • 애니메이션의 이름, 실행시간은 필수다.
  • 호모데우스 미래의 역사, 김대식의 인간 vs 기계
  • 무한반복 : animation-iteration-count: infinite
  • transition-timing-function property
  • jade, pug / sass, less
    header, footer 코드 조각을 미리 만들어놓고 include시켜서 개발
    css는 원래 변수 쓰거나 연산이 불가능 한데 전처리기를 쓰면 가능해진다.

  • form / fields / legend
    html 4.01에서는 필수가 아닌데 xhtml에서는 필수이다. html5는 다 된다.(??)
    fieldset : 연관성 있는 폼 관련 요소들을 묶어준다.
    legend : 필드셋이 하는 일을 명시한다.

  • input 태그에 title 달면 부연설명되고 마우스 올리면 뜬다.

  • autofocus는 별로 권장하지 않는다.
  • polyfill은 지원하지 않을 때의 대안책을 의미한다.
  • input type submit / button type submit
