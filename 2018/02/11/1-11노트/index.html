1.11노트

2018-02-11
  • WIFI(Wireless Fidelity)
  • 802.11 != wifi
    802.11 : IEEE에서 개발된 표준 무선통신기술
    wifi : 와이파이 얼라이언스의 상표. 802.11 기술을 사용하는 무선 근거리 통신망 제품
  • 2.4GHz 주파수는 파장이 길기 때문에 장애물을 통과할 때 일어나는 반사와 굴절 능력이 5GHz보다 좋고 수신 거리도 길다.
  • 5GHz 주파수는 대역폭이 넓어서 채널끼리 서로 간섭을 일으킬 확률이 낮고 한 번 연결되면 빵빵한 속도를 자랑한다.
    wifi 2.4g vs 5g
  • WIFI는 고주파수 대역을 ac로 쓰는게 좋다.
  • 1대1 연결들을 어떻게 조합할지를 고민한 것이 Network topology(망구성방식/topology:위상수학)
  • star, bus가 많이 쓰임
  • 우리 컴퓨터에서 구글 본사까지의 연결은 유선 연결이다.
  • OSI 7 layer
  • Presentation layer - OS
  • UTF-8
  • Session layer
  • 사용자의 연결 상태를 서버가 갖고 있으면 세션, 웹브라우저가 갖고 있으면 쿠키 (ex : 항공사 홈페이지 재접속시 가격 인상, 네이버 서버의 세션이 30분동안 로그인 유지)
  • Transport layer
  • 용량 큰 파일을 패킷으로 쪼갠다. 받는 쪽도 Transport layer
  • 방화벽은 화이트리스트 시스템이다.
  • 인터넷은 교통, 이더넷은 도로로 치환해서 이해하면 된다. (도로 말고도 해로 항공로를 통해 교통을 이용할 수 있다.)
  • 이더넷은 LAN에서 많이 활용되는 기술 규격이며 OSI 7 Layer에서 Data-link layer에 위치한다.
  • Network layer
  • Physical layer는 실제 데이터가 왔다갔다 하는 물리적인 레이어이다.
  • Data-link layer는 케이블과 케이블 사이를 이어준다.
  • 테더링은 일대일, 핫스팟은 와이파이 ??
  • 폴링은 기술이름 에이잭스는 도구이름
  • Hypertext를 전송하기 위한 프로토콜이 http이다.
  • FTP : File Transfer Protocol / SMTP : mail
  • Internet : 나를 어떻게 규정할 것인가, 어떤 자료를(application) 어떻게 보내는데(transport) 나는 어떤 사람인가(주소체계를 어떤걸 쓸건지) ??
  • TCP/IP : 주소체계와 어떻게 보낼지가 정해져있다.
  • request head에 국가코드가 들어있다.
  • TCP 패킷 : STREAM, 다 받아야 읽기 가능 / UDP : DATAGRAM, 10개 중 한 개만 받아도 읽기 가능 (가래떡 예시)
  • 포트 번호는 1024 위쪽으로 써라
