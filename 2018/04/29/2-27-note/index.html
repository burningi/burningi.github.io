2018/2/27 note

2018-04-29
  • 객체인데 순회가능한 객체가 이터러블
  • 이터러블 : String, arguments, HTMLCollection, Nodelist, map, set, array
  • map, set 이런것들을 퉁쳐서 list 또는 collection이라고 부른다
  • concat이나 spread둘다 빠르다
  • timer함수는 자신을 나타내는 id를 반환한다(clearInterval)
  • proto 직접 수정은 금지
  • 자바스크립트의 대부분은 객체이다. primitive 제외한 나머지는 다 객체다
  • 따라서 객체지향형 언어다.
  • 퍼포먼스 좀 저하되도 그거 상쇄할만한 편리함을 주면 쓰게 된다.(es6 class)
  • class Person 찍어보면 function으로 나온다
  • class 첫문자열은 대문자
  • es6 클래스 내부에는 메소드만 있어야된다.
  • property는 반드시 constructor안에서 선언
  • 클래스도 호이스팅이 일어난다.(선언식이니까)
  • es6는 선언식 이전에 참조하면 에러난다 (템포러리 데드존 /일시적 사각지대)
  • 선언에서 초기화까지의
